# Support

## Getting Help

Use GitHub Issues for:

- bugs
- documentation problems
- feature requests
- workflow clarification

When opening an issue, include:

- local OS
- Python version
- local `rsync` version if relevant
- remote OS details if relevant
- exact command used
- exact error output

## What To Expect

This project is maintained as a focused operational tool. Support is best-effort and works best when reports are reproducible and specific.
